# MSDB: Madurai Smart Drop Booth 🚛♻️

### **The IoT-Enabled Smart Waste Ecosystem**
MSDB is a B2C (Business-to-Consumer) solution designed for Madurai to incentivize waste segregation at the source through a mobile, solar-powered smart van.

## 🚀 Overview
Traditional waste collection lacks public motivation. MSDB solves this by rewarding citizens with **Green Credits** for every kg of segregated waste they drop into our smart booth.

## 🛠️ Tech Stack
* **Brain:** Raspberry Pi 4 / Arduino
* **Language:** Python
* **Sensors:** Load Cells (Weight), Ultrasonic (Level Detection)
* **Backend:** REST API for real-time Cloud Sync
* **Power:** 100W Solar Panel Integration

## 🏗️ Hardware Architecture
1. **Van Height:** 5 Feet (Prototyped)
2. **Slots:** Dual-compartment (Dry & Wet)
3. **Display:** 7-inch Touch Interface for QR and Point display.



## 💡 Key Features
* **Instant Gratification:** Points credited immediately via API.
* **Anti-Fraud:** AI-based weight anomaly detection to prevent malpractice.
* **Sustainability:** 100% off-grid power via solar panels.

## 📊 Business Model
* Selling high-quality segregated waste to industrial recyclers.
* Redeemable 'Green Tokens' for local Madurai shops.
